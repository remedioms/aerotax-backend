┃ 1. KERNSATZ — Klassifikation nach FollowMe-Standard                      ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

Die ENTSCHEIDENDE Frage pro Tag/Tour ist nicht "Inland oder Ausland?",
sondern: "GIBT ES EINE ÜBERNACHTUNG IM AUSLAND?"

DREI HAUPT-KATEGORIEN:

┌─────────────────────────────────────────────────────────────────────────┐
│ A) SAME-DAY-TAGESTRIP (= Z72) — Heimkehr am gleichen Tag                │
│                                                                          │
│ User fliegt morgens raus, kehrt abends zurück nach Homebase.            │
│ Egal ob Ziel Inland (HAM/MUC/STR) ODER EU-Ausland (CPH/BER/AMS/VIE):   │
│ → Z72 mit 14€ Inland-Tagestrip-Pauschale (LH-/FollowMe-Konvention)     │
│ → 1 Fahrtag, 1 Arbeitstag, 0 Hotelnächte                                │
│ → Voraussetzung: Gesamtabwesenheit ≥ 8h (mit Briefingzeiten prüfen)    │
│                                                                          │
│ BEISPIEL: 11.01 LH882 A FRA→CPH 06:30 + LH883 E CPH→FRA 18:45          │
│         → Same-Day → Z72 mit 14€ (NICHT Z76 weil keine Übernachtung)   │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ B) INLAND-MEHRTAGES-TOUR (= Z73 + Arbeitstage) — Hotel im Inland        │
│                                                                          │
│ User fliegt zu Inland-Ziel, übernachtet dort, kehrt zurück.             │
│ Z.B. Schulung in München mit Hotel-Übernachtung.                        │
│ → Anreise-Tag = Z73 (14€)                                               │
│ → Volltage zwischen An/Ab = nur Arbeitstage (kein zusätzliches VMA)    │
│ → Abreise-Tag = Z73 (14€)                                                │
│ → 1 Fahrtag (Tour-Start), n Hotelnächte                                  │
│                                                                          │
│ BEISPIEL: 04-08.03 D4 SCHULUNG in MUC mit Hotel                         │
│         → 04.03 Z73 (Anreise), 05+06+07.03 Arbeitstage,                │
│           08.03 Z73 (Abreise) = 2× Z73 = 28€                            │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ C) AUSLANDSTOUR (= Z76) — Übernachtung im AUSLAND                       │
│                                                                          │
│ User fliegt zu Auslands-Ziel, übernachtet dort, kehrt zurück.           │
│ → ALLE Tage der Tour = Z76 (auch An-/Abreise mit FRA-Stempel)           │
│ → BMF-Auslands-Anreise-Pauschale für Tag 1 + letzten Tag                │
│ → BMF-Auslands-24h-Pauschale für Volltage zwischen An/Ab                │
│ → 1 Fahrtag pro Tour, n Hotelnächte (FL-Marker)                          │
│                                                                          │
│ BEISPIEL: 03-06.05 BLR-Tour (Indien)                                    │
│         → 03.05 BMF Indien-Anreise 30€                                  │
│         → 04.05 BMF Indien-24h 39€                                      │
│         → 05.05 BMF Indien-24h 39€                                      │
│         → 06.05 BMF Indien-Anreise 30€                                  │
│         → Z76 = 30+39+39+30 = 138€ für diese Tour                       │
│         → 3 Hotelnächte (03-05.), 1 Fahrtag, 4 Arbeitstage              │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ D) Z74 (24h Inland) — sehr selten                                        │
│                                                                          │
│ Inland-Tag mit ZW=12 ohne Ab/An-Zeiten — User war ganzen Tag dienstlich │
│ in Inland aber NICHT Tagestrip-/Anreise-Pattern.                        │
│ → 1 Tag × 28€                                                            │
│                                                                          │
│ Beispiel: D4 SCHULUNG mit ganztägiger Anwesenheit ohne Übernachtung,    │

═══ DENK-SCHEMA PRO TAG ═══

  Schritt 1: Welche Aktivität laut Marker?
             FREI/U/K → kein Arbeitstag, fertig.
             SBY/RES → Arbeitstag, kein Fahrtag, fertig.
             EK/EM/EH (Recurrent/Schulung) → AT + FT, ABER: prüfe Schritt 1b.
             D4/DD (mehrtägige Schulung) → IMMER Schritt 1b prüfen.
             Tour (A/E/FL) → weiter Schritt 2.

  Schritt 1b (NEU — KRITISCH für Z73): Schulung mit Hotel-Übernachtung?
             Folgt direkt am nächsten Tag wieder Schulung+ohne FREI dazwischen?
             ODER: ist Schulungs-Ort NICHT FRA/MUC/HAM/Homebase (z.B. D4 in
             externem Center)? ODER: gibt es Hotel-Anweisung/Layover-Ortswechsel?
             → JA → Mehrtages-Inland-Tour → Z73 für Anreise + Abreise!
                    Volltage dazwischen = AT ohne zusätzliches VMA.
             → NEIN → reine Office-Schulung am Homebase, kein VMA.

             FAUSTREGEL: Wenn 3+ aufeinanderfolgende EM/EH/D4-Tage OHNE FREI
             dazwischen, ist es FAST IMMER eine Schulung mit Hotel (Z73-Tour).
             FollowMe-Konvention: lieber Z73 als Office annehmen wenn unsicher.

  Schritt 2: Bei Tour (A/E/FL): Wo ist die Übernachtung?
             a) Z72 — Same-Day-Tagestrip — STRENGES HARD-GATE!
                Z72 darf NUR vergeben werden wenn ALLE Bedingungen ERFÜLLT sind:
                  ✓ A-Marker UND E-Marker am SELBEN Kalendertag
                  ✓ KEIN FL-Marker am Tag oder am Folgetag
                  ✓ KEINE Tour-Fortsetzung am Folgetag (kein A am nächsten Tag,
                    kein RES/SBY direkt anschließend)
                  ✓ KEIN Hotel- oder Layover-Indiz im Einsatzplan
                  ✓ Gesamtabwesenheit > 8h (Briefing+Block+Sign-Off+Anfahrt)
                Wenn EINE Bedingung fehlt → NICHT Z72 → prüfe Z73 oder Z76!
                Egal ob Ziel CPH/AMS/VIE oder MUC/HAM/STR — gleiche Hard-Gate-Regel.
             b) Hotel-Marker (FL) am Folgetag UND Routing/Layover-Ort = INLAND?
                → Z73 für Anreise+Abreise (14€ pro Tag), Volltage = Arbeitstage.
                Inland-Routing erkennst du an Layover-Code aus der Inland-Liste:
                FRA, MUC, HAM, DUS, STR, CGN, HAJ, BER (TXL/SXF), LEJ, NUE,
                BRE, FMO, PAD, NRN, FKB, HHN, SCN, DRS, ERF, FDH, RLG, KSF.
             c) Hotel-Marker (FL) am Folgetag UND Routing-Layover = AUSLAND?
                → Z76 für ALLE Tour-Tage (BMF-Land-Pauschale für An/Ab + 24h
                  für Volltage).

  ═══ Z72-HARD-GATE BEISPIELE ═══

  ZULÄSSIG Z72:
    • 11.01: A LH882 FRA→CPH 06:30 + E LH883 CPH→FRA 18:45 + 12.01 FREI
      → Same-Day, kein FL, kein Folgetag-Dienst → Z72 ✓

  NICHT Z72 (= Z73 oder Z76):
    • 11.01: A LH900 FRA→MUC 16:00 + E LH901 MUC→FRA 21:00 + 12.01 EM Schulung
      → Auch wenn 11.01 Same-Day wirkt: Folgetag Schulung ohne FREI dazwischen
        deutet auf zusammenhängenden Dienst → KEIN Z72.
        Wenn Schulung in MUC: ganze Sequenz Z73. Wenn Schulung FRA: Office.
    • 04.03: A LH950 FRA→MUC 08:00 + E LH951 MUC→FRA 22:00 + 05.03 D4 SCHULUNG MUC
      → 05.03 zeigt User ist STILL in MUC → 04.03 war Anreisetag → Z73, nicht Z72.

  REFLEX-FEHLER VERMEIDEN:
    Wenn Pass 1 zu viel Z76 ergeben hat: NICHT pauschal Tage in Z72 verschieben.
    Sondern: Schau jede einzelne Tour an, prüfe Z72-Hard-Gate, sortiere richtig.

  ═══ MULTI-STOP-TOUREN MIT GEMISCHTEN LAYOVERN — TAG-FÜR-TAG ═══

  Eine Tour darf NICHT pauschal komplett als Z76 klassifiziert werden, nur weil
  irgendwo im Verlauf ein Auslands-Layover vorkommt. Bei Multi-Stop-Touren mit
  gemischten Inland- und Auslands-Layovern muss PRO KALENDERTAG entschieden werden.

  Entscheidend pro Kalendertag:
    1. Wo endet der dienstliche Tag?
    2. Wo schläft der User?
    3. Ist der Übernachtungsort Inland oder Ausland?
    4. Gibt es am selben Tag eine Heimkehr aus einer vorherigen Tour plus
       neuen Same-Day?

  REGEL:
    • Übernachtung im Inland → Z73 für An-/Abreisetag der Inland-Übernachtung.
    • Übernachtung im Ausland → Z76 nach BMF-Auslandspauschale.
    • Same-Day-Turnaround OHNE Übernachtung → Z72, NUR wenn eindeutig A+E am
      selben Tag und keine Tourfortsetzung.
    • Ein Kalendertag kann fachlich eine Mischsituation enthalten — z.B.
      Rückkehr aus mehrtägiger Tour + neue Same-Day-Tätigkeit. Dann NICHT
      pauschal nur Z72 setzen, sondern die vorherige Tour korrekt abschließen
      und die neue Aktivität separat dokumentieren.

  BEISPIEL-SCHEMA (gemischte Multi-Stop-Tour):
    Tag 1: FRA→<Inland-Stadt>, Übernachtung Inland       → Z73 Anreise
    Tag 2: <Inland>→<Ausland>, Übergang aus Inland-Layover → Z73 Abreise (von
           Inland) bzw. neuer Tour-Start ins Ausland prüfen
    Tag 3: <Ausland>→<anderes Ausland>, Übernachtung      → Z76 Volltag
    Tag 4: Auslands-Layover                              → Z76 24h
    Tag 5: Rückkehr Homebase                             → Z76 Abreise

  ANTI-PATTERN (häufiger Bug):
    Multi-Stop FRA→<Inland>→<Ausland>→<Ausland>→FRA pauschal als komplett
    Z76 abrechnen, obwohl Tag 1 eine Inland-Übernachtung ist. Das verschiebt
    Z73-Tage in Z76 und führt zu Inkonsistenz mit den AG-Auslandsspesen.

  Z73 ist NICHT auf reine Inlandstouren beschränkt!
    Z73 entsteht IMMER wenn an einem Kalendertag eine INLAND-Übernachtung
    stattfindet — egal ob:
    • Reine Inland-Tour (FRA→MUC mit Hotel zurück nach FRA)
    • Reine Inland-Schulung (D4 in MUC mit Hotel mehrtägig)
    • Multi-Stop-Tour mit Inland-Layover (FRA→BER→ZAG→FRA: Tag 1 = Z73 BER)
    • Tour-Mix: User kehrt aus Auslandstour zurück + neue Tour beginnt am
      gleichen Tag mit Inland-Layover

    Wenn dein Klassifikations-Output Z73 = 0 ergibt UND es im Dienstplan
    Indizien für Inland-Übernachtungen gibt (z.B. mehrtägige EM/D4 außerhalb
    Homebase, Inland-Layover-Codes in Multi-Stop-Touren), hast du Z73 übersehen.

  HOTELNÄCHTE — DETERMINISTISCH AUS FL-MARKERN:
    hotel_naechte = Σ aller FL-Marker im Dienstplan, EGAL ob Inland oder Ausland.
    Eine FL-Markierung im Dienstplan IST eine Hotelnacht — die Land-Frage ist
    erst für die Z73-vs-Z76-Klassifikation relevant, nicht für die Hotelnacht-
    Zählung an sich.

    Häufiger Bug: hotel_naechte zählt nur Auslands-Hotels. Das ist falsch.
    Inland-Hotels (Schulungs-Hotels MUC/HAM/BER, Inland-Tour-Layover) zählen
    GLEICHWERTIG. Trinkgeld-Pauschale 3,60€/Nacht gilt für jede Hotelnacht.

  ═══ Z72-HARD-GATE — VOLLSTÄNDIGE BEDINGUNGSLISTE ═══

  Z72 darf vergeben werden, wenn:
    ✓ A-Marker UND E-Marker am selben Kalendertag
    ✓ KEINE FL-/Hotel-/Layover-Übernachtung an diesem Tag
    ✓ Der Tag gehört nicht zu einer am Vortag begonnenen mehrtägigen Tour
      (kein offener Layover aus der vorherigen Nacht — User schlief zuhause)
    ✓ Der Tag startet keine mehrtägige Tour (kein Übernachtung-am-Folgetag)
    ✓ Abwesenheit > 8h plausibel

  WICHTIG — Z72 ist NICHT verboten nur weil Vortag dienstlich war:
    ✗ FALSCHE Auslegung: "Kein vorheriger oder folgender Tag darf Dienst sein"
    ✓ KORREKT: User kommt aus mehrtägiger Tour zurück (Tag X = Heimkehr) und
      macht am Tag X+1 einen sauberen Same-Day-Trip (A+E am Tag X+1, keine
      Übernachtung Tag X+1) → Tag X+1 IST Z72.

  Mischfall-Regelung (Heimkehr + Same-Day am selben Tag):
    Wenn am SELBEN Kalendertag Heimkehr aus mehrtägiger Tour UND ein
    Same-Day-Trip stattfindet:
    → NICHT pauschal nur Z72.
    → Im tage_detail beide Komponenten getrennt aufführen:
      • Tour-Abschluss: Z76- oder Z73-Abreisetag (je nach letzter Übernachtung)
      • Same-Day-Aktivität separat dokumentieren (zählt aber typisch nicht als
        zusätzliches Z72, weil der Tag bereits VMA aus der Tour-Abreise hat)

  REFLEX-FEHLER VERMEIDEN (beide Richtungen):
    ❌ "Pendel" Pass-1: zu viele Z72 → Pass-2 reduziert auf 0 = FALSCH
    Z72 ist eine REGULÄRE Klasse für Same-Day-Tagestrips. Eine Vollzeit-Crew
    hat typisch mehrere echte Same-Day-Trips pro Jahr. Wenn dein Output
    Z72 = 0 ergibt UND es klare A+E-am-selben-Tag-Sequenzen ohne FL gibt,
    hast du das Hard-Gate zu strikt interpretiert. Korrigiere zurück.

  ═══ MEHRTÄGIGE SCHULUNGEN — Z73 STATT OFFICE ═══

  Wenn 2+ aufeinanderfolgende Schulungs-/Office-Tage (EH/EM/EK/D4/DD) vorliegen
  UND der Ort nicht eindeutig tägliche Homebase-Anfahrt ist, MUSS aktiv auf
  Inland-Übernachtung geprüft werden.

  Bei Hotel-/Layover-/auswärtigem Schulungsort im Inland:
    • erster Tag → Z73 Anreise (14 €)
    • letzter Tag → Z73 Abreise (14 €)
    • Zwischentage → Arbeitstag (kein zusätzliches VMA), ggf. Z74 nur bei
      nachgewiesener 24h-Inland-Abwesenheit ohne An-/Abreise-Charakter

  NICHT automatisch als Office klassifizieren wenn:
    • Schulung mehrtägig
    • Schulungs-Ort in Einsatzplan ≠ Homebase
    • Hotel-Indikator vorhanden ODER keine tägliche Heimfahrt-Indikation

  Schritt 3: Bei Z76: BMF-Auslands-Pauschale für das Land aufschlagen
             (Anreise-Satz für Tag 1 + letzten Tag,
              24h-Satz für Volltage)

═══ DECISION-TREE Z72 vs Z73 vs Z76 (KRITISCH) ═══

  Frage 1: Hotel-Marker (FL) im Dienstplan an diesem Tag?
    NEIN → Wenn Marker A+E am gleichen Tag → Z72. Sonst kein VMA.
    JA → Frage 2.

  Frage 2: Was ist der Layover-Ort (3-Letter-Code im Einsatzplan)?
    Code in Inland-Liste oben → Z73 für An/Abreise.
    Code NICHT in Inland-Liste → Z76 für gesamte Tour.
    Code nicht eindeutig → konservativ Z73 (Inland) annehmen, in unklare_tage
    eintragen.

  Frage 3 (Sanity-Check): Z77 in der SE für diesen Tag stfrei-Inland oder Ausland?
    stfrei-Inland (FRA/Homebase) + Auslands-Layover → trotzdem Z76 (LH zahlt
                                                       FRA-stempel aber Tour
                                                       war Ausland → Wissens-
                                                       Fehler #1)
    stfrei-Ausland → Z76 zwingend (LH bestätigt Auslandstour)

  Z73 = 0 ist UNWAHRSCHEINLICH bei Vollzeit-Crew (aber theoretisch möglich).
  Wenn deine Klassifikation auf Z73=0 endet, prüfe nochmal alle EM/EH/D4-Blöcke
  und potenzielle Inland-Layovers — häufiger Bug ist mehrtägige Schulung mit
  Hotel als "Office daily" zu verbuchen. Wenn nach erneuter Prüfung wirklich
  keine Inland-Übernachtung im Dienstplan steht: dann eben Z73=0, das ist OK.

§ 9 ABS. 1 Nr. 4 EStG — FAHRTEN WOHNUNG ↔ ERSTE TÄTIGKEITSSTÄTTE:
  - Entfernungspauschale 0,30 €/km bis 20. km, 0,38 €/km ab 21. km
  - Einfache Strecke × Anzahl Fahrtage (NICHT × 2)
  - 1 Tour = 1 Fahrtag (egal wie lang)
  - Mehretappen ohne Heimkehr (FRA→GVA→OTP→FRA) = 1 Fahrtag total

§ 9 ABS. 4a EStG — VERPFLEGUNGSMEHRAUFWAND (VMA):
  Inland (BMF-Pauschalen, fix):
    - Tagestrip > 8h ohne Übernachtung (Z72): 14 €/Tag
    - An-/Abreisetag mit Übernachtung (Z73):  14 €/Tag
    - 24h-Tag mit Übernachtung (Z74):         28 €/Tag
  Ausland (BMF-Tagessätze pro Land — siehe Tabelle Abschnitt 5)

§ 9 ABS. 1 Satz 3 Nr. 6 EStG — REINIGUNG BERUFSKLEIDUNG:
  - Pauschal 1,60 €/Arbeitstag (BFH-Praxis bei einheitlicher Uniform)
  - Anwendbar bei LH-Crew (Uniform)

REISENEBENKOSTEN TRINKGELDER:
  - Pauschal 3,60 €/Hotelübernachtung (BMF-Verwaltungspraxis)

§ 3 Nr. 16 EStG — STEUERFREIE ARBEITGEBER-ERSTATTUNGEN:
  - Z77 (Σ stfrei vom AG aus SE) mindert NUR den Reisekosten-Topf (Z72-Z76)
  - NICHT die Reinigungs-/Trinkgeld-/Fahrtkosten-Töpfe
  - Wenn Z77 > VMA-Pauschalen-Summe: Reisekosten-Topf bleibt 0,
    Überschuss ist im Brutto regulär versteuert (kein Schaden)

§ 3 Nr. 16 EStG — Z17/Z18 IN LSB:
  - Z17 (AG-Fahrkostenzuschuss steuerpflichtig) → mindert NUR Fahrtkosten-Topf
  - Z18 (15% pauschal versteuerter AG-Fahrkostenzuschuss) → Indikator für
    AG-Jobticket: User darf KEINE ÖPNV-Kosten zusätzlich absetzen

┃ 3. EASA-FTL — EU 965/2012 (Hotelnacht-Erkennung)                         ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

ORO.FTL.105 — Definitionen:
  - Home Base: Ort wo Crew üblicherweise für eine FDP einsteigt (hier FRA)
  - Reporting Time: 60 min vor STD (Standard Time of Departure)
  - FDP (Flight Duty Period): vom Reporting bis 30 min nach Block-On
  - Layover: Aufenthalt am Zielort mit Möglichkeit zur ortsfesten Ruhe (Hotel)
  - Turnaround: kein Layover, Crew bleibt im Flughafen-Bereich

ORO.FTL.235 — Rest Periods:
  - Mindest-Ruhe AT HOME BASE: 12 Stunden
  - Mindest-Ruhe AWAY FROM HOME BASE: 10 Stunden inkl. Hotel-Schlafzeit
  - Crew Rest Compartment während Flug zählt NICHT als Rest-Period

ORO.FTL.205 — Maximum FDP:
  - Abhängig von Reporting-Zeit und Anzahl Sectors
  - Standard 13h, gestreckt bis 14h bei limitierten Sectors

KONSEQUENZ FÜR HOTELNACHT-ZÄHLUNG:
  → Eine Hotelnacht zählt NUR wenn ≥10h Bodenzeit am Zielort
  → FL-Marker bei LH = echter Layover ≥10h → 1 Hotelnacht
  → Nachtflug-Heimkehr (raus 22:00, zurück 05-06:00 nächsten Tag) =
    Turnaround, KEIN Hotel
  → Bei Mehrtages-Auslandstour: jeder Zwischen-Tag = 1 Hotelnacht
    Tag-Beispiel JNB 4-Tage:
      Tag 1: A FRA→JNB → 1 Hotelnacht
      Tag 2: FL JNB    → 1 Hotelnacht
      Tag 3: FL JNB    → 1 Hotelnacht
      Tag 4: E JNB→FRA → KEINE Hotelnacht (Heimkehr)
      Total: 3 Hotelnächte

┃ 7. BRIEFINGZEITEN LH-Kabine (für Z72-Tagestrip-8h-Schwelle)              ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

Ein Inland-Tagestrip qualifiziert für Z72 (Pauschale 14€) wenn die Gesamt-
Abwesenheit von der Wohnung ≥ 480 Min (8h) beträgt.

  Abwesenheit = Anfahrt + Briefing + Block-Time + 30 min Sign-Off + Anfahrt

Briefing-Zeit:
  • Wenn explizit im Einsatzplan/Dienstplan: diese benutzen
    (Format: "Briefingzeit(LT FRA): 03/02/25 20:10")
  • Sonst Faustregel:
    → Kurzstrecke (Block ≤ 4h):  85 Min vor STD (1:25 h)
    → Langstrecke (Block > 4h): 110 Min vor STD (1:50 h)

Sign-Off: 30 Min nach Block-In (einheitlich für alle Tour-Typen)

Anfahrt:
  • User-Input wenn vorhanden (Form-Feld 'anfahrt_min')
  • Sonst aus km × 1,5 Min/km abgeleitet
  • Default 30 Min wenn unbekannt

Beispiel: Tagestrip CPH mit Block 1:30 + 27km Anfahrt
  → Briefing 85 Min + Block 90 Min + Sign-Off 30 Min = 205 Min
  → Plus 2× Anfahrt 41 Min = 82 Min
  → Total Abwesenheit 287 Min < 480 Min → KEIN Z72-Tag

VOLLZEIT-CREW (Quote ~100%, ganzes Jahr aktiv):
  Arbeitstage:    grob 110-170 (Tour + Office + Standby + Schulung)
  Fahrtage:       grob 50-100 (1 pro Tour-Start + tägliche Office-Anfahrt)
  Hotelnächte:    grob 40-100 (alle FL-Marker = Auslands-Layovers)
  Z72-Tage:       0-50 (Same-Day-Trips, stark crew-/airline-abhängig)
  Z73-Tage:       1-20 (Inland-Schulungen mit Hotel + Inland-Layover)
  Z74-Tage:       0-3 (sehr selten — 24h Inland ohne Tagestrip-Pattern)
  Z76 (€):        meist der GRÖSSTE WK-Posten bei aktiver Crew
                  (Auslandstouren × BMF-Pauschalen)
  Z77 (€):        typisch 3.000-7.000 (LH stfrei gezahlte Spesen)

TEILZEIT/MUTTERSCHUTZ/KRANKHEIT-Anteil:
  → Alle Werte oben proportional skalieren (z.B. 60%-Quote → ~60% der
    Bandbreiten). NIEMALS aus Bandbreite zurückrechnen — IMMER aus
    Dienstplan zählen.

HARTE INVARIANTEN + AUDIT-PLAUSIBILITÄT:
  • Z76 > Z77 ist KEINE harte mathematische Unmöglichkeit, aber ein
    starkes Audit-Warnsignal: SE-Vollständigkeit, Auslands-/Inland-
    Klassifikation, Storno-Zeilen, Zwölftel/Kürzungen und Mahlzeiten prüfen.
  • Hotelnächte ≤ Arbeitstage (harte logische Invariante)
  • Fahrtage ≤ Arbeitstage (harte logische Invariante)
  • Z76-Tage entsprechen ungefähr Σ(Auslands-Hotelnächte + 1 Anreise/Tour)
  • Auslandsspesen-Summe (SE) ≈ Z76 (BMF) ±30% — sonst Klassifikations-Bug

BEOBACHTUNGEN AUS DER PRAXIS (statistisch, nicht zwingend):
  • Z73 = 0 ist unwahrscheinlich bei Vollzeit — meistens hat Crew
    mind. 1 Inland-Schulung mit Hotel pro Jahr (Recurrent in MUC/HAM)
  • Z76 in € ist normalerweise der größte WK-Posten (>1.000€ bei Vollzeit
    mit Auslandsanteil), weil Langstreckenflüge die Pauschalen hochtreiben
  • Hotelnächte und Fahrtage liegen meist im Verhältnis 1:1 bis 1.5:1

→ Wenn deine Klassifikation diese Bandbreiten/Checks verletzt:
  STOP, geh zurück zu den Files, lies und zähl nochmal.
→ Wenn deine Klassifikation in den Bandbreiten liegt: das ist KEIN Beweis
  dass es stimmt — nur dass es plausibel groß ist.
